## 如何使用（How to use）

**首先 :**

直接创建一个 `resume.json` 文件在自己的 special 仓库下.

**最后**

访问 https://visiky.github.io/resume?user=ApocalypsezZ&branch=master

参数说明:

| 参数   | 描述          | 默认值       |
| ------ | ------------- | ------------ |
| user   | github 用户名 | 必选         |
| template | 模板        | 默认: template1 |
| branch | 分支名        | 默认: master |
| mode | 模式        | 备注: 默认为‘只读’模式，设置为: `mode=edit` 即可进入编辑模式 |
| lang | 语言        | 默认: zh_CN |
